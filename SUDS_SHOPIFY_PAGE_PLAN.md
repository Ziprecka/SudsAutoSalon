## Manual Shopify settings

- Navigation: set main menu to Home, Services, Service Areas, Reviews, About, Contact, Book Now.
- Book Now should link to `/collections/mobile-details` unless you later create a dedicated booking page.
- Contact should link to `/pages/contact-us`.
- Add Instagram URL in theme social settings if missing.
- Add real Google/Judge.me review excerpts before enabling review schema.
- Ensure product images have descriptive alt text such as `Signature mobile detail interior result by Suds Auto Salon`.
- Keep checkout products in the Mobile Details collection, but use landing pages for SEO and education.
